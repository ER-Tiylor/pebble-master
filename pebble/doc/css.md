
## 使用 padding 实现图片自适应并保持宽高比
``` scss
.img-wrapper{
  position: relative;
  width: 100%;
  padding-bottom: 66%;
  img{
    position: absolute;
    width: 100%;
    height: 100%;
  }
}
```
